# Music + Terminal?
(only works for linux!)

yep!
how to use:
  1. extract the zip
  2. run ```./CliMusic``` in the terminal
  3. read the readme further!

Navigation:

  Up/Down Arrows - navigation Up/Down
  
  Enter          - Start Selected Song Playback
  
  Escape         - Stop Playback

  Q              - Quit

How to add custom music (or random files :3, it should work): (tested on .txt files, binary files)

  1. drag your files into the music folder
  2. start CliMusic
  3. Enjoy!
